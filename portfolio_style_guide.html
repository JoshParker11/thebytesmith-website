Typography

Font Families

Inter (Primary)

Clean, modern sans-serif for body text, UI elements, and general content. Professional and highly readable.

JetBrains Mono

Monospace font for code blocks, technical content, and data display. Developer-focused with excellent readability.

Crimson Text

Elegant serif for blog posts, articles, and longer form content. Adds warmth and readability to prose.

Text Sizes

Extra Small (0.75rem) - Captions, footnotes
Small (0.875rem) - Secondary text, metadata
Base (1rem) - Body text, standard content
Large (1.125rem) - Emphasis, lead paragraphs
Extra Large (1.25rem) - Subheadings
2XL (1.5rem) - Section headers
3XL (1.875rem) - Page titles
4XL (2.25rem) - Hero headings

Buttons

Usage Guidelines

  • Primary: Main CTAs, important actions (Deploy, Publish, Save)
  • Secondary: Supporting actions (Edit, View, Cancel)
  • Outline: Tertiary actions, toggles
  • Ghost: Subtle actions, navigation elements

Forms

Design Principles

Visual Hierarchy

  • Use gradient text for main headings
  • Amber (#D87C2A) for section headings
  • Moss (#4C9A2A) for subheadings
  • Muted gray for secondary information

Spacing & Layout

  • Consistent 8px grid system
  • Generous whitespace for readability
  • Card-based layouts for content organization
  • Responsive grid systems

Interaction Design

  • Subtle hover animations (translateY, glow effects)
  • Glass morphism with backdrop blur
  • Focus states with amber accent rings
  • Smooth 0.3s transitions

Content Tone

  • Professional yet approachable
  • Technical without being intimidating
  • Outdoor/nature-inspired metaphors
  • Growth and learning mindset
